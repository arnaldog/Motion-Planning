\section{Experimentos}
Los experimentos son realizados sobre un conjunto de instancias que caracterizan situaciones de diferentes dificultades para el agente. Uno de lo objetivos de realizar la experimentación sobre distintas instancias es el de poder encontrar parámetros adecuados o criterios ad-hoc para cada tipo de instancia (nivel de dificultad) y así establecer un razonamiento adecuado para la futura utilización del algoritmo (heurísticas). Otro objetivo muy importante es medir la calidad de las soluciones encontradas y poder hacer un breve análisis de éstas conjunto con los tiempos de ejecución del algoritmo y evaluar el tiempo de convergencia respecto a los parámetros. 

La búsqueda de parámetros adecuados significa realizar análisis exploratorio de datos. Para analizar los datos, pueden ser utilizadas diferentes técnicas, una técnica a considerar es estudiar los datos desde la perspectiva de un modelo, donde los parámetros del problema son las variables explicativas (o características) del modelo y medidas como el fitness o el tiempo de ejecución son la salida del modelo.

La idea de estudiar el fitness explicado a través de los parámetros puede ser considerada porque la relación entre ambos no es fuertemente causal, dado que el algoritmo inicializa pseudo aleatoreamente a las partículas y existen incertidumbres asociadas al valor final de la función objetivo lo que permite realizar análisis estadístico para estudiar tal situación.

Con un modelo estadístico asociado al problema es posible realizar simulaciones y así ampliar la perspectiva desde el punto de vista de las ejecuciones dado que la obtención de un conjunto variado de soluciones constituye consumir largos períodos de tiempo.

Es importante señalar que el objetivo del análisis no es el de determinar o proponer un modelo de regresión u otra clase de ajuste, sino el de estudiar el comportamiento generando modelos empíricos como las redes neuronales artificiales.

\subsection{Organización de los Experimentos}
Se realiza por cada instancia una serie de experimentos probando diferentes parámetros, midiendo la calidad de las soluciones y observando el tiempo de ejecución del algoritmo. Otro dato importante de observar es la convergencia del valor de la función objetivo durante las iteraciones del algoritmo lo que permite ver a posteriori el comportamiento de la velocidad del enjambre de partículas.
